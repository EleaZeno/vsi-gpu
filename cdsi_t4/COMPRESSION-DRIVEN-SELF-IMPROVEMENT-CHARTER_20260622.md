# 路线宪法：压缩率驱动的开放域自改进 (Compression-Driven Open-Domain Self-Improvement, CDSI)

**日期**: 2026-06-22 · 作者: 李子轩 + QClaw · 状态: 纲领（路线宪法，后续所有实验对照此文，防止飘回中庸刷分）

---

## 0. 一句话命题
> 一个**微型模型**，在**没有标准答案的开放域**（讲故事/对话），以"对自己见过数据的**无损压缩率**"为**唯一内生奖励**，**递归地**自我改进——不仅压缩数据，还压缩"自己的压缩策略"。验证它能否在开放域达成 DeepSeek 们只在数学域达成的"**非饱和自改进**"，且用 **unseen-G** 证明它是**真理解**（压缩率在没见过的数据上也涨）而非**自欺**（死记训练集）。

---

## 1. 第一性定律推导链（今天 6-22 完整思想链，勿丢）

### 定律一：智能 = 压缩（有严格数学）
- DeepMind 2023《Language Modeling Is Compression》(arXiv 2309.10668)：预测模型 ≡ 无损压缩器，数学等价。
- 源头：Solomonoff 归纳 / 柯氏复杂度 / AIXI（Hutter）。最强智能 = 找到生成数据的**最短程序**。
- 推论：loss↓ = 压缩比↑ = 理解加深，三者同一件事。

### 定律二：强 = "该压的压，不该压的别压"
- RNN/SSM 死因：把**历史**也强压进固定状态 → 信息瓶颈 → 角色串戏（我们擂台 m1/m2 实证）。
- Transformer 胜因：**知识压进权重（泛化），历史原样留 KV-cache（不压，精确回看）**。把两件事分开。
- 我们的 GMW 判死刑：想用一个机制把历史也压了 → 违背此定律 → 连续两轮垫底。

### 定律三：望远镜终点 = 递归自压缩
- MDL 隐藏的递归：压数据 → 压"压缩器" → 压"压缩压缩器的规律"… 人类智能即如此分层。
- Transformer 只停在"第0层事实 + 半个第1层模式"，**没有"对自己压缩结果再压缩"的机制** → 不会真正举一反三/提炼世界观。
- **变聪明的两条路**：①喂更多数据（大厂烧钱）。②把已有知识压得更短（不需新数据）。
- **第二条路 ≡ 自进化的数学本质**：递归对自己理解再压缩 = 不接触新数据就变聪明（人"想通"/AlphaZero 互搏机制）。

## 2. 查重结论（2026-06-22，诚实切边界）

### 已被做（必须避开）
| 已有工作 | 是什么 |
|---|---|
| 自蒸馏 (Phi-3) | 压缩**知识**，不靠外部教师 |
| 程序归纳 (DreamCoder) | 学单个程序 |
| 自适应深度/KV压缩 (MoR NeurIPS25, MoD, AdaKV) | 为**省算力**的工程优化 |
| **DeepSeekMath-V2 (2025-11)** | 生成器+验证器+元验证防幻觉自改进闭环——**但只在"对错可判定"的数学域，用大模型大算力** |

### 真无人区（精确坐标）
> **大厂的自改进靠"外部可判定奖励"（数学对错/围棋胜负）。进入"无标准答案的开放域"，自改进就失去锚——怎么知道是改进还是自欺？**
> **压缩率是唯一不需要外部答案、纯内生、有数学保证的客观信号。** 能把见过的东西压得更短（且无损还原）= 真的更理解了，无需任何人判对错。
- 没人把"压缩率"当**开放域自改进的内生奖励**。
- 没人**递归压缩"压缩策略本身"**。
- 没人在**微型规模**上验证这条（大厂往大模型+数学域走，因为好刷榜）。
- 理论 Solomonoff/MDL 指明了 60 年，因不好刷榜没人碰。

## 3. 为什么这条统一了我们所有的牌
- **VSI 自进化** = CDSI 的自改进循环本体。
- **κ / Variance-Inequality (2512.02731)** = "压缩压坏（自欺）"的临界点理论；压太狠丢真相，κ 告诉你临界在哪。
- **盲评方法论** = 开放域质量的旁证（压缩率是主信号，盲评防"压短了但变成乱码"）。
- **unseen-G 防作弊探针** = 防自欺锚（真规则在没见过的数据上也成立 = 压缩率也涨；死记不涨）。
- **微型模型** = 载体；用"把知识压成精炼规则"补"参数少"，一条好规则顶一万条相关性 = 窄域碾压的物理机制。
- **假胜利免疫系统** = 全程纪律，防止把"压短=变乱码"误判成胜利。

## 4. 铁律（继承用户科研纪律）
1. 等参数/等算力公平对比。
2. 盲评不看架构名。
3. unseen-G 防自欺：压缩率必须在**没见过的数据**上也涨，否则判 [CHEAT]（死记）。
4. 多 seed 钉死（≥2-3），单 seed 形状清晰也不算坐实。
5. 不为好看硬凑，诚实证伪比假胜利值钱；但也别拿证伪当挡箭牌——拿经检验的正结果。
6. 复用已验证世界模块（vsi_lm_nl_world 等），不重造轮子防漂移。
